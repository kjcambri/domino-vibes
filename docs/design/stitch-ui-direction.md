# Stitch UI Direction

Sprint 18 translates the Stitch concepts into one cohesive direction for Domino Vibes: **Island Elite Social Club**.

## Direction

The UI should feel like a premium Caribbean/Latino domino hall: deep emerald backgrounds, polished mahogany framing, felt table surfaces, cream dominoes/text, luminous gold actions, tropical teal presence, and soft glow shadows. It should feel social and competitive without becoming a casino interface.

## Concept Sources

- Design 1 influenced the landing page atmosphere: strong brand identity, palm-shadow/felt mood, gold hero actions, and a live table preview.
- Design 2 influenced the lobby: premium table cards, active/rejoin banner, social hub layout, table status clarity, and global chat placement.
- Design 3 influenced the game room: board-first HUD, side panel for social/game state, bottom rack/action feel, and stronger current-turn clarity.

## Forbidden Language

Avoid gambling/casino framing:

- High Rollers
- Buy-in
- Prize Pool
- cash values
- betting/wagering
- casino chips or poker-style language

Use product-safe language:

- Featured Table
- Live Match
- Weekly Challenge
- Bragging Rights
- Cutthroat 4
- Points to Win: 6
- Private Tables, Coming Soon
- Ranked, Coming Soon
- Community Tournament, Coming Soon

## MVP vs Future Concepts

Current playable MVP:

- Cutthroat 4
- lobby/table/game chat
- rejoin active games
- secure hidden hands
- server-saved board geometry
- next-round and game-over flow

Future concept cards must be visibly disabled or marked “Coming Soon” and must not be wired to gameplay.

## Typography

The Stitch direction recommends Epilogue for headings and Be Vietnam Pro for body text. Sprint 18 keeps the existing font stack to avoid adding external font loading and layout risk during production testing. A future font pass can add self-hosted or carefully preloaded fonts.
